## Ìºü Key Features

- **Real-time Collaboration**: Multiple users can edit documents simultaneously
- **Comments System**: Position-based comments with threading and mentions
- **@Mentions**: Mention users in documents and comments
- **Rich Text Editing**: Powerful editor with Tiptap and extensive formatting
- **Firebase Integration**: Secure authentication and data storage
- **Galaxy Theme**: Beautiful, custom-designed cosmic interface
- **WebRTC & WebSocket**: Dual real-time communication systems
- **Responsive Design**: Works perfectly on all devices

## Ìª† Tech Stack

### Frontend
- **React 18** - Modern React with hooks and context
- **Vite** - Lightning-fast development server
- **Tiptap** - Headless rich text editor
- **Socket.io Client** - Real-time communication
- **Firebase SDK** - Authentication and data
- **Custom CSS** - 100% custom galaxy-themed design

### Backend
- **Node.js + Express** - RESTful API server
- **Socket.io** - Real-time WebSocket server
- **Firebase Admin** - Authentication and Firestore
- **JWT** - Secure token-based authentication
- **WebRTC Support** - Peer-to-peer collaboration

### Database
- **Firebase Firestore** - NoSQL document database
- **Firebase Authentication** - User management
- **Firebase Storage** - File uploads (optional)
